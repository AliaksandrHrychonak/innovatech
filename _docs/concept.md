# Концепт

## Структура сайта

### Главная страница — Hero Section

- **Полноэкранный видео-фон | Анимация:** Живые кадры современных теплиц, автоматизация, рост растений в ускоренной съемке
- **Минималистичный заголовок, например** "Будущее агротехнологий. Сегодня." с крупной типографикой
- **Плавная прокрутка:** Parallax-эффекты при скролле вниз
- **Ключевые цифры:** Анимированные счетчики — количество проектов, площадь теплиц, экономия ресурсов

### Навигация — Интуитивная и визуальная

- **Sticky-меню:** Прозрачное меню с эффектом затемнения при прокрутке
- **Визуальное меню категорий:** Крупные карточки с иконками и hover-эффектами для каждого направления
- **Хлебные крошки:** Всегда видимый путь навигации
- **Умный поиск:** Предиктивный поиск с автодополнением и визуальными подсказками

### Страницы по направлениям — Storytelling подход

- **Каждое направление = отдельная история:** Начинается с проблемы, показывает решение
- **Крупные изображения продуктов:** 360° обзор, возможность zoom, видео работы оборудования
- **Интерактивные схемы:** Как работает технология — анимированные диаграммы
- **Кейсы клиентов:** До/После с конкретными цифрами ROI

### Детальные страницы продуктов — Максимум визуала

- **Галерея:** Листание изображений с плавными переходами
- **Технические характеристики:** Иконки + короткие пояснения, разворачиваются по клику
- **3D-модели:** Возможность повернуть и рассмотреть оборудование со всех сторон
- **Сравнение моделей:** Таблица сравнения с подсветкой преимуществ
- **Конфигуратор???**
- **Техническая документация при скачивании**

### Страница "Проекты" — Визуальное портфолио

- **Masonry-сетка:** Разноразмерные карточки проектов с фильтрацией
- **Фильтры:** По типу теплиц, культурам, масштабу, географии
- **Детальные кейсы:** Фотографии высокого качества, видео-туры, отзывы клиентов
- **Карта проектов:** Интерактивная карта с метками реализованных объектов

### 11.9 Конфигуратор проекта — Интерактивный инструмент

- **Пошаговый мастер:** Выбор типа теплицы → культуры → оборудования → расчет стоимости
- **Сохранение конфигурации:** Возможность скачать PDF или отправить на email
- **Запрос коммерческого предложения:** Прямо из конфигуратора

### 11.11 Мобильная версия — First approach

- **Адаптивный дизайн:** Все функции доступны на смартфоне
- **Быстрые действия:** Кнопки "Позвонить", "Написать", "Рассчитать стоимость"
- **Оптимизированный контент:** Сокращенные тексты, вертикальные изображения
- **Быстрая загрузка:** Progressive Web App с оффлайн-доступом к ключевым разделам

### 11.12 Технические фишки стиля

- **Плавные анимации:** 60 FPS, transitions на всех элементах
- **Типографика:** Крупные заголовки, достаточно white space
- **Цветовая палитра:** Ограниченная, премиальная (белый, темно-серый, акцентный зеленый)
- **Фотографии:** Только высокого качества, профессиональные, с эмоцией
- **Микроинтеракции:** Hover-эффекты, плавные появления элементов при скролле
- **Загрузка контента:** Lazy loading с плейсхолдерами

### 11.13 Доверие и социальное доказательство

- **Логотипы клиентов:** Крупные бренды на главной
- **Награды и сертификаты:** Визуально выделенный блок
- **Статистика удовлетворенности:** NPS-score, процент повторных заказов

## 💡 12. Ключевые принципы реализации

### 12.1 Что должно быть "на глазах" сразу

- **Ценностное предложение:** Что получит клиент — экономия, урожайность, автоматизация
- **Социальное доказательство:** Цифры, логотипы, отзывы в первом экране
- **Призыв к действию:** Яркая кнопка "Рассчитать проект", "Консультация", "Смотреть кейсы"
- **Доступность:** Контакты, чат, телефон всегда видны

### 12.2 Пользовательский опыт

- **Максимум 3 клика до цели:** Любая информация доступна быстро
- **Понятная иерархия:** От общего к частному
- **Контекстные подсказки:** Tooltips, пояснения терминов
- **Персонализация:** Запоминание предпочтений пользователя

### 12.3 Производительность

- **Скорость загрузки <2 секунд:** Оптимизация изображений, CDN, кэширование
- **SEO-оптимизация:** Семантическая разметка, быстрая индексация
- **Аналитика:** Отслеживание поведения пользователей для улучшения
